Strategy Hub

Dealhound One place for research outlines, positioning drafts, and decisions.

Each card below is a self-contained output (research, positioning, content plan, etc.). Open one, make decisions per item (Adopt / Adapt / Skip), add notes, then click Generate feedback prompt at the bottom to produce a message you can paste back to Claude for the next deliverable. Decisions are persisted locally in your browser.
